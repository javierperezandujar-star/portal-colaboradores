🚀 CÓMO USAR (PASOS SIMPLES):
═════════════════════════════════════════════════════════════════

1️⃣  LEE PRIMERO:
    → GOOGLE_DRIVE_SETUP.md (SUPER IMPORTANTE)
    
2️⃣  CONFIGURA GOOGLE CLOUD:
    → Ve a: https://console.cloud.google.com
    → Crea proyecto
    → Habilita Google Drive API
    → Obtén Client ID y API Key
    
3️⃣  EDITA src/utils.js (línea ~20):
    apiKey: "TU_API_KEY_AQUI",
    clientId: "TU_CLIENT_ID.apps.googleusercontent.com",
    
4️⃣  INSTALA Y EJECUTA:
    npm install
    npm run dev
    
5️⃣  ABRE EN NAVEGADOR:
    http://localhost:3000
    
6️⃣  CONECTA CON GOOGLE:
    Click en "Conectar con Google Drive"
    
7️⃣  ¡LISTO! Los datos se guardarán automáticamente en Drive
